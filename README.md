# Task Manager

## Project Overview
The Task Manager is a simple application that allows users to manage their tasks efficiently. It provides an intuitive interface for adding, updating, and deleting tasks. This documentation covers the project's structure, features, and installation instructions.

## Features
- **Add Task**: Users can create new tasks with a title and description.
- **Update Task**: Users can edit existing tasks.
- **Delete Task**: Users can remove tasks that are no longer needed.
- **Task List View**: All tasks can be viewed in a list format.
- **Filter Tasks**: Users can filter tasks based on their status (e.g., completed, pending).

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Avnavteotia/Task-Manager.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Task-Manager
   ```
3. Install the requirements:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
Once the application is running, users can access it via `http://localhost:3000`. The application interface allows you to:
- Add new tasks by filling out the form on the main page.
- Edit existing tasks by clicking on the edit button next to each task.
- Delete tasks by clicking on the delete button.
